Full Query Pipeline

python3 query.py "What was Tesla's revenue in 2024?"
  → Dense search: ChromaDB top-20 by cosine similarity
  → BM25 search: keyword match top-20
  → RRF merge: combine both lists → 40 candidates
  → Reranker: cross-encoder scores all 40 → keeps top-5
  → Parent lookup: swap child chunks → full parent sections
  → GPT-4o: question + 5 parent sections → answer
  → RAGAS: score retrieval + faithfulness + correctness
  → Logger: print to console + write full trace to logs/

All Decisions + Why

Decision Why
Hierarchical chunkingWord-count chunking splits mid-sentence and merges different sections. Paragraph-level child chunks preserve meaning; section-level parent chunks preserve context.
BM25 hybrid searchEmbeddings miss exact terms (numbers, abbreviations). BM25 misses paraphrase. Together they cover both failure modes.
Reciprocal Rank FusionMerging dense + sparse lists without needing to tune score thresholds. Works by rank position, not raw scores.
Local cross-encoder rerankerSimilarity measures shape of meaning. Reranker reads question + chunk together and scores actual answer relevance. Free, local, inspectable.
Parent lookupRetrieve small (precise match), pass large (full context). Child chunks are precise enough to find; parent sections are rich enough for GPT-4o to reason over.
Terminal script, no UIA UI hides scores and internals. Terminal output exposes every score at every step — essential for learning what each component contributes.
Console + JSON loggingConsole for immediate human feedback. JSON for querying across runs later — "which questions had low retrieval scores?"
RAGAS evaluationWithout measurement we can't know if a change helped. Three independent scores expose exactly which layer failed: retrieval, faithfulness, or correctness.

The learning loop this enables

Run all 20 golden dataset questions → get RAGAS scores → pick the lowest-scoring question → read its JSON log → see exactly which component failed → change that one thing → run again → compare scores. Every improvement is measurable. Every failure is diagnosable.
